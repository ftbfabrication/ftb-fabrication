FTB Fabrication Website - Starter Files

Files included:
- index.html: the website content and structure
- style.css: the design, colors, layout, and mobile styling

How to edit:
1. Open index.html in Notepad or VS Code.
2. Replace your-email@example.com with your real email.
3. Replace Your phone number with your phone number, or remove that line.
4. Replace the photo placeholder sections later with real photos.

Cloudflare Pages easiest upload method:
1. Zip these files or drag the folder into Cloudflare Pages if direct upload is available.
2. Set the custom domain to ftbfabrication.com and www.ftbfabrication.com.

GitHub method:
1. Create a new GitHub repository.
2. Upload index.html and style.css.
3. In Cloudflare Pages, connect the GitHub repository.
4. Build command: leave blank
5. Output directory: / or leave blank
